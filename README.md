## Como Rodar
- [Deploy](https://flask-poke-trader.herokuapp.com/)
- <em>Usando Python:</em>
  - $ pip install -r requirements.txt
  - $ export FLASK_APP=app.py
  - $ export FLASK_ENV=development
  - $ flask run
